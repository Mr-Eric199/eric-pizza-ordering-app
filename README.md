# 🍕 Eric Pizza Ordering App

A simple pizza ordering web application built with PHP and MySQL, using Materialize CSS for styling.

## 🚀 Features
- Add new pizzas with title, email, and ingredients
- View a list of all available pizzas
- View detailed information about each pizza
- Delete pizzas with confirmation popup
- Responsive design with Materialize CSS

## 🛠️ Technologies Used
- PHP
- MySQL
- Materialize CSS
- JavaScript

## ⚙️ Setup & Installation
1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/eric-pizza.git
   ```
2. **Move into the project folder**
   ```bash
   cd eric-pizza
   ```
3. **Set up MySQL database**
   - Create a new database `eric_pizza`
   - Import the `eric-pizza.sql` file
   - Update `config/db_connection.php` with your database credentials

4. **Run the project using XAMPP (or any local server)**
   - Start Apache and MySQL in XAMPP
   - Open `http://localhost/eric-pizza` in your browser

## 📌 Usage
- Click **"Add a Pizza"** to add a new pizza
- Click on a pizza to view more details
- Click **"Delete"** to remove a pizza (confirmation required)
